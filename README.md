# Secure Azure Docker Blog

## Overview

This project demonstrates the end-to-end deployment of a Dockerized blog application on Microsoft Azure, with a strong focus on security and cloud best practices.

The application is containerized using Docker and deployed on Azure infrastructure, then secured using SSL/TLS, Web Application Firewall (WAF) rules, and geo-location access restrictions.

---

## Tech Stack

* Microsoft Azure
* Docker
* SSL/TLS Encryption
* Web Application Firewall (WAF)
* Geo-location Filtering
