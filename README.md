# Secure Azure Docker Blog

## Overview
This project demonstrates the end-to-end deployment of a Dockerized blog application on Microsoft Azure, with a strong focus on security and cloud best practices.

The application is containerized using Docker and deployed on Azure infrastructure, then secured using **Azure Key Vault, SSL/TLS, Web Application Firewall (WAF) rules, and geo-location access restrictions**.

---

## Tech Stack
- Microsoft Azure  
- Docker  
- SSL/TLS Encryption  
- Azure Key Vault  
- Web Application Firewall (WAF)  
- Geo-location Filtering  

---

## Project Workflow

### Day 1 – Deployment
- Created and configured Azure resources  
- Built and deployed Docker container  
- Hosted and customized the blog application on Azure  

### Day 2 – Security Implementation
- Configured **Azure Key Vault** for storing secrets and certificates  
- Created and analyzed **self-signed and trusted SSL/TLS certificates**  
- Applied Azure App Service Managed Certificates  
- Strengthened application security  

### Day 3 – Access Control & Testing
- Configured **Azure Web Application Firewall (WAF)** and applied custom rules  
- Implemented **geo-location-based access restrictions**  
- Reviewed and remediated **Security Center recommendations**  
- Tested application security and verified end-to-end deployment
